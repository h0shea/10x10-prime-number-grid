Welcome!

10x10 Prime Number Grid is a website that generates 10x10 number grids and detects numbers that are primes in the grid. This website can also generate grids that are beyond from the 1-100 grid, meaning that you could discover more primes over 100.

Instructions

When starting the website, the website would create a grid from 1-100. To move the grid, click the '>>' button to go to the next grid, and click the '<<' button to go to the previous grid. You could also go to the specific grid you want by typing the ordinal number of that grid into the insert box (e.g. inputting "256" will lead to Grid #256 for numbers from 25601 to 25700) and press the 'Enter' button. The 'Reset' button is used to go back to Grid #0, and the 'Randomize Grid' button is used to go to any random grid (from Grid #0 to Grid #10000). You could also press one of the numbers of the grid to see its description card (its label is "Card #n"). The card will give some descriptions about the number (e.g. if the number is prime or not, how many factors does the number has, etc).
